# Railway / Render deploy config

# Pentru Railway:
# 1. Mergi la railway.app
# 2. New Project → Deploy from GitHub
# 3. Urca folderul w213cloud pe GitHub
# 4. Railway detecteaza automat Node.js si face deploy
# 5. Settings → Generate Domain → copiaza URL-ul

# Pentru Render:
# 1. Mergi la render.com  
# 2. New → Web Service → conecteaza GitHub
# 3. Build Command: npm install
# 4. Start Command: node server.js
# 5. Copiaza URL-ul generat

# Variabile de mediu (optional):
# PORT - setat automat de Railway/Render
